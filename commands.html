Commands

Moderation Commands

  • ban (aliases: b) - Ban a user from the server
  • kick - Kick a member from the server
  • timeout (aliases: mute) - Timeout/mute a user
  • unban (aliases: ub) - Unban a user
  • warn (aliases: w) - Warn system with subcommands:
    • warn - Issue a warning
    • warn list - List warnings for a user
    • warn delete - Delete a specific warning
    • warn clear - Clear all warnings
    • warn punishment - Configure punishment
    • warn punishment list - List punishments
    • warn punishment reset - Reset punishment configuration

Moderation Logging

  • modlog (aliases: mod) - Moderation log system:
    • modlog setup - Initial setup
    • modlog channel - View current log channel
    • modlog channel set - Set the log channel
    • modlog toggle - Enable/disable logging
    • modlog event - Configure event logging
    • modlog case - View a specific case
    • modlog user - View all cases for a user

Utility Commands

  • alias - Command alias system
    • alias add - Create a command alias
    • alias remove - Remove an alias
    • alias list - List all aliases
    • alias clear - Clear all aliases
  • prefix (aliases: p) - Prefix management
    • prefix add - Add a server prefix
    • prefix remove - Remove a server prefix
    • prefix list - List all prefixes
    • prefix self - Set personal prefix
    • prefix self toggle - Toggle personal prefix
  • invoke (aliases: inv) - Auto-response system
    • invoke set - Set an auto-response
    • invoke preview - Preview an invoke
    • invoke reset - Reset an invoke
    • invoke clear - Clear all invokes
    • invoke list - List all invokes
    • invoke variables - Show available variables
  • welcome (aliases: welc, greet, join) - Welcome message system
    • welcome list - List welcome messages
    • welcome remove - Remove a welcome message
    • welcome clear - Clear all welcome messages
    • welcome variables - Show available variables
    • welcome preview - Preview a welcome message
    • welcome move - Move welcome to different channel
  • role - Role management
    • role - Add/remove roles from members
    • role create - Create a new role
    • role delete - Delete a role
  • crypto - Cryptocurrency information
    • crypto - Get crypto price info
    • crypto convert - Convert between cryptocurrencies
    • crypto price - Get price for specific amount
  • Safety & Filtering
    • disable (aliases: dis, restrict, off)
      • disable - Disable a command for user/channel
      • disable list - List all disabled commands
      • enable - Re-enable a disabled command
    • filter (aliases: automod)
      • filter add - Add a filtered word
      • filter list - List filtered words
      • filter remove - Remove a filtered word
      • filter regex add - Add regex filter
      • filter regex list - List regex filters
      • filter regex remove - Remove regex filter
      • filter invite on - Enable invite filtering
      • filter invite off - Disable invite filtering
      • filter invite edit - Edit invite filter settings
